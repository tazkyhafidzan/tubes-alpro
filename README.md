# APLIKASI MANAJEMEN DAN REVIEW CO-WORKING SPACE
Created by Tazky Hafidzan & Ratna Bariatutamimah

Aplikasi ini adalah program berbasis antarmuka baris perintah (**CLI - Command Line Interface**) yang dirancang untuk mengelola data operasional tempat kerja bersama (*co-working space*) serta memfasilitasi interaksi antara pemilik tempat dan pengguna (pekerja remote atau *freelancer*).

Program ini bertindak sebagai platform dua arah. Di satu sisi, pemilik *co-working space* dapat mengelola katalog properti mereka. Di sisi lain, pengguna dapat mencari ruang kerja yang ideal berdasarkan preferensi mereka, sekaligus memberikan transparansi kualitas layanan melalui sistem ulasan (*review*) dan rating digital.

## 🎯 Tujuan Program

1. **Manajemen Data Dinamis (Sisi Pemilik):** Memberikan sarana bagi pemilik usaha untuk melakukan operasi CRUD (*Create, Read, Update, Delete*) pada data *co-working space* mereka secara mandiri dan akurat.
2. **Efisiensi Pencarian (Sisi Konsumen):** Membantu pekerja remote dan *freelancer* menemukan tempat kerja yang sesuai dengan kebutuhan produktivitas mereka secara cepat.
3. **Transparansi Layanan:** Menyediakan platform ulasan dan rating objektif untuk membantu pengguna lain dalam memilih ruang kerja terbaik.
4. **Implementasi Algoritma:** Menerapkan konsep algoritma pengurutan dan pencarian fundamental pada studi kasus nyata.

---

## 🛠️ Fitur Utama & Spesifikasi

Aplikasi ini mengimplementasikan struktur data terstruktur serta beberapa algoritma fundamental pemrograman sesuai spesifikasi berikut:

* **Manajemen Data & Ulasan (CRUD):** * Menambahkan, mengubah, dan menghapus data *co-working space* (nama, lokasi, fasilitas, harga sewa).
  * Menambahkan, mengubah, dan menghapus ulasan serta rating skala tempat yang dikunjungi.
* **Fitur Pencarian (Searching):** Mencari *co-working space* berdasarkan nama atau lokasi menggunakan metode **Sequential Search** dan **Binary Search**.
* **Fitur Pengurutan (Sorting):** Menampilkan daftar tempat berdasarkan harga sewa termurah atau rating tertinggi menggunakan algoritma **Selection Sort** dan **Insertion Sort**.
* **Fitur Penyaringan (Filtering):** Menampilkan daftar *co-working space* yang memiliki fasilitas spesifik tertentu (seperti *WiFi, meeting room, atau private desk*).
